# <img src="https://upload.wikimedia.org/wikipedia/commons/a/a9/Amazon_logo.svg" width="100"> Amazon Product Discount & Rating Analysis Dashboard

## 📌 Project Overview
This project analyzes Amazon product data to understand patterns related to product discounts, ratings, and customer reviews. The dashboard helps explore product categories, brand popularity, and discount distribution using interactive visualizations.

The dashboard is created using Microsoft Power BI.

---

## ❗ Problem Statement
E-commerce platforms such as Amazon contain thousands of products across different categories and brands, each with different prices, discounts, ratings, and customer reviews. Due to the large volume of product data, it becomes difficult to easily identify patterns such as which categories offer higher discounts, which brands receive more customer engagement, and how product ratings relate to customer reviews.

---

## 📂 Dataset
The dataset contains information about Amazon products including:

- Product Name  
- Brand Name  
- Main Category  
- Actual Price  
- Discounted Price  
- Discount Percentage  
- Rating  
- Rating Count (Customer Reviews)

---

## 📈 Dashboard Features

### 🔹 KPI Cards
- Total Products
- Total Reviews
- Average Discount
- Average Rating

### 🔹 Visualizations
- Brand Reviews
- Product Category Distribution
- Discount by Category
- Discount vs Reviews
- Product Details Table

### 🔹 Interactive Filter
- Category slicer for filtering the dashboard

---

## 🛠 Tools Used
- Microsoft Power BI  
- Power Query (Data Cleaning)  
- DAX (Data Analysis)
